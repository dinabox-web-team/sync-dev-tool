# sync-dev

Ferramenta simples para sincronizar um diretório local com um servidor remoto via SSH (rsync) com opção de monitoramento em tempo real.

Para a documentação completa e instruções passo a passo, visite a documentação hospedada (GitHub Pages):

https://dinabox-web-team.github.io/sync-dev-tool

Para instalar rapidamente a versão estável mais recente (branch `main`):

```bash
# Baixa o script da branch `main` e executa a instalação (requer sudo)
curl -L https://raw.githubusercontent.com/dinabox-web-team/sync-dev-tool/main/sync-dev.sh -o sync-dev.sh
sudo bash sync-dev.sh --install
```

(Ver documentação para opções avançadas, verificação de integridade e recomendações de segurança.)
