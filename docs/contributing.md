---
title: Contributing
---

Contribuições são bem‑vindas! Sugestões:

- Abra uma branch a partir de `main` para mudanças pequenas/rápidas e crie um Pull Request.
- Inclua descrição clara do problema e como testar.
- Execute `bash -n sync-dev.sh` e, se disponível, `shellcheck sync-dev.sh` antes de submeter.

Opcional: adicionar um GitHub Action que rode `shellcheck` e `bash -n` em PRs para manter qualidade.
